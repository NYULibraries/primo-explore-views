Welcome to the new Bobcat!

This version of the new BobCat is customized for Cooper Union and is being tested during Summer 2018. You will see similar versions at NYU and at other participating libraries.

Please use the feedback form to send comments and questions. Your input is greatly appreciated!

As an alternative, use BobCat Standard, but keep in mind that some features are only available when using the new interface.

 Books & More

Enter search terms, then choose from the drop-down menu either Consortium Catalog (to search all participating libraries) or Cooper Library Catalog. If you do not make a choice the default is all participating libraries.

Please note that results displays for electronic resources may vary between this Cooper version of BobCat and the versions at NYU libraries or others. Contact a reference librarian for assistance.

 Searching for Articles

Do not use this site to search for individual articles within newspapers, magazines, proceedings, etc.

Instead, use the article databases found on the main Cooper Library website under Resources by Subject.

Contact a Reference Librarian for further assistance.
