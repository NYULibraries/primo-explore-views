Explanation of Request Options

Eligible borrowers may see one or more of the request options below.

For more information on each option, please visit the Libraries Collection Access page (opens in a new window) on the library website.

  • Request
    • Locker delivery for pick-up at Bobst and Dibner Library
    • Home delivery available in the continental U.S. 48
    • Office delivery (Faculty only)
    • Pick-up at IFA or ISAW Libraries
  • Request Scan: Electronically delivered scans of portions of physical materials

If the item is not available at one of our libraries, eligible borrowers will be redirected to request through Inter-Library Loan (ILL)(opens in a new window) .
