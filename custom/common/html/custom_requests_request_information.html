How do I get this item and how long will it take?

  • If you want a book now, and it is "Available", find the item on the shelf and check it out at the Circulation Desk. Use our call numbers and floor maps resource for help.
  • If you can wait 3-5 days, request the item for locker pick-up.
  • If you only need 1-2 chapters of a book, request a digital scan.
