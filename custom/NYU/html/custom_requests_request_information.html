Explanation of Request Options

Eligible borrowers may see one or more of the delivery options below. These are available only for physical materials available at one of the NYC library locations.

For more information on each option, please visit the Libraries Collection Access page (opens in a new window) on the library website.

  • Request
    • Locker delivery for pick-up at Bobst and Dibner Library
    • Digital scanning of physical materials, electronically delivered
    • Home delivery available in the continental U.S. 48
    • Office delivery (Faculty only)
  • Request Inter-Library Loan (ILL): Request a book, article, or chapter from another library.
