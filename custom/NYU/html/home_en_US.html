NYU Returns & Collections Access
  • Access to in-copyright material in the HathiTrust emergency digital collections will end on Friday, August 20, followed by resumption of in-person access to print collections on Monday, August 23.
  • Physical library materials can be requested for in-person pick-up, delivery, or digital scan. Request options (“Request / Request Scan”) are found on the details page of a particular item.
  • If a resource is not available at one of our libraries, E-Z Borrow and Inter-Library Loan can be used to request books from partner libraries.

Updates & New Features
  • The library has created quick, 2-minute online tutorials for searching the catalog. These include tips on searching, scanning, and getting library resources.
  • Students and instructors can search for collections that are digital and multi-use using the Always Available Online filter.
  • Instructors can add “Always Available Online” items to Course Reserves in one-click via Add to Course Reserves (Instructors Only) link.

Need Help?

Use Ask A Librarian or the “Chat with Us” icon at the bottom right corner for help starting a search or understanding collections available.
